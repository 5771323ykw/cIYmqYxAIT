# 项目简介

## 核心功能
- 管理员用户界面
- 用户管理
- 部门管理
- 文件类型管理
- 文件信息管理
- 系统管理
- 个人中心

## 适用场景
- 企业内部数据资产管理
- 数据资产登记与追踪
- 数据安全与合规性监控

## 技术特点
- B/S架构，易于部署和使用
- MVC模式，代码结构清晰
- 采用Java、SpringBoot、Vue等技术栈，保证系统的稳定性和扩展性
- 支持IDEA、JDK1.8、Mysql、Maven、Navcat等主流开发环境
